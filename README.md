# Open PoL

Public website for Open PoL: open proof-of-location ideas, capture-time provenance, and witness-machine infrastructure.

The site is intentionally static and GitHub Pages friendly. It can be served directly from the repository root.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Source Material

The first version draws language and positioning from Eduardo Brito's Proof-of-Location grant and research notes in `/home/eduardo/personal/other/grants`.

The layered Proof-of-Location figure comes from `/home/eduardo/projects/PHD/the-thesis/assets/figures/pol-layers.pdf` and is included as both `assets/pol-layers.pdf` and `assets/pol-layers.png`.
